## 🌟 Overview

**Agnes Catering** is a premier culinary service provider based in **Westlands, Nairobi**. We bridge traditional Kenyan heritage (such as Swahili Pilau, Nyama Choma, and Mukimo) with contemporary fine dining aesthetics and white-glove hospitality. 

This repository contains the complete source code for our modern web platform, featuring responsive cross-gadget design (mobile to laptop), glassmorphic UI/UX, live movement animations, WhatsApp direct integration, and PWA support for instant installation.

---

## 🚀 Key Features

- **🍽️ Curated Culinary Menus**: Explore traditional Kenyan delicacies, international fusion, artisan breakfasts, tropical desserts, and signature drinks.
- **📱 PWA & Cross-Gadget Ready**: Fully optimized for mobile phones, tablets, and desktop laptops with Progressive Web App support for easy installation.
- **✨ Enhanced Glassmorphism & Live Motion**: Smooth frosted glass panels, ambient shadows, and live floating/glow keyframe animations.
- **💬 Direct WhatsApp Booking**: Instant booking and inquiry integration across all menu items, services, and galleries.
- **📸 Authentic Event Photography**: Real event and dish photographs showcasing our live cooking stations, buffet spreads, and pastry displays.

## 🛠️ Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone the repository:
   ```bash
   git clone https://github.com/ismahdeismail-beep/agnes-cutlery.git
   cd agnes-cutlery
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
